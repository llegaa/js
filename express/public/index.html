Версия V2

Есть 3 страницы отдающие ответы: Главная, Статистика, Комментарии

Главная Статистика Коментарии

Главная

Доступна по адресу /api/v2/

принимает GET запросы. На любой запрос отвечает "Hello".

Не важно тело запроса.

Статистика

Доступна по адресу /api/v2/stats

Принимает GET запросы. Введёт статистку запросов.

Отправляет json вида {"user-agent": count_request}

Не важно тело запроса.

Комментарии

Доступна по адресу /api/v2/comments

Принимает get запрос => возвращает все элементы комментариев

Принимает post запрос => добавляет элементы body в формате json { "name": "myname", "text":"mytext" }

по адресу /api/v2/comments/:id

Принимает get запрос в параметре id передаётся id модели => возвращает одну модель или ошибку

Есть 3 страницы отдающие ответы: Главная, Статистика, Комментарии

Главная Статистика Коментарии

Главная

Доступна по адресу /api/v1/

принимает GET запросы. На любой запрос отвечает "Hello".

Не важно тело запроса.

Статистика

Доступна по адресу /api/v1/stats

Принимает GET запросы. Введёт статистку запросов.

Отправляет json вида {"user-agent": count_request}

Не важно тело запроса.

Комментарии

Доступна по адресу /api/v1/comments

Принимает POST запросы.

В теле запроса отправляем обычный текст, который будет записан в массив.

Отправляет полный массив со всеми комментариями.

Если в теле запроса ничего не будет, страница вернёт текущий массив комментраиев.
